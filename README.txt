Mailchecker
===========================================================================================================

MailChecker is a simple tool to check your mail in an easy way from the command line. It uses Lesnikowski's
Mail.dll email component (all rights reserved to its owner), give credits at http://www.lesnikowski.com/.

Usage: mailchecker emailaddress password [imapserver]

emailaddress	The e-mail address to check the e-mails from.
password	Password to acces to the given e-mail address.
[imapserver]	The imap server where the e-mail account is. Optional, if not provided will
		use Gmail imap server.
